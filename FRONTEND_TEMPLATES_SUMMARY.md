## 🚀 快速开始

### 1. 访问销售单列表
```
http://localhost:5000/sales/list
```

### 2. 创建新销售单
```
点击"新建销售单"按钮 → 填写表单 → 动态添加商品 → 提交
```

### 3. 查看销售单详情
```
点击销售单号 → 查看流程进度 → 导航到分配单
```

### 4. 创建分配单
```
在销售单详情中点击"进行分配" → 选择策略 → 确认分配
```

### 5. 执行拣货
```
进入拣货单详情 → 逐项点击"完成拣货" → 库位自动转移
```

### 6. 执行出库
```
进入出库单详情 → 点击"出库"按钮 → 输入出库数量 → 确认
```

## ✨ 最佳实践

### 1. 响应式设计
所有模板采用Bootstrap栅格：
- 桌面端：3列 → 1列布局
- 平板端：2列 → 单列布局
- 手机端：完全单列布局

### 2. 易用性设计
- **清晰的导航**：返回按钮和面包屑
- **状态可视化**：Badge和进度条
- **快速操作**：模态框减少页面跳转
- **实时反馈**：JavaScript计算实时显示

### 3. 数据安全
- 危险操作确认（删除、拒绝）
- 权限检查（某些操作仅限于特定状态）
- 输入验证（必填字段、数量范围）

### 4. 性能优化
- 分页限制列表大小
- 表格响应式容器避免横向滚动
- 模态框减少DOM节点

## 🎓 模板使用指南

### 如何修改样式
编辑 `base.html` 的 `<style>` 部分，所有模板会自动继承。

### 如何添加新字段
1. 在对应的视图函数中传递新数据
2. 在模板中使用 `{{ variable_name }}` 显示

### 如何添加新路由
创建新的模板文件 → 在视图中定义路由 → 在导航中添加链接

### 如何调试模板
1. 启用Flask调试模式
2. 检查浏览器开发者工具
3. 查看Flask日志输出
4. 使用Jinja2模板引擎调试器
